README.md
=========

Build: `sbcl --load main.lisp build`, this will generate an executable file `slider`.

Play slideshow(test.lisp): `./slider test.lisp`

Key:

* j: next
* k: previous
* R: reload
* q: quit

Dependency:
* sbcl
* Quicklisp
* cl-ncurses(libncursesw.so.5)
* uffi
